Ficha Técnica

Título: Predictive modeling and web-based tool for cervical cancer risk assessment: A comparative study of machine learning models
Autores: Ritu Chauhan, Anika Goel, Bhavya Alankar, Harleen Kaur
Resumen: En respuesta al crecimiento rápido de bases de datos en la era digital, se ha desarrollado CHAMP (Cervical Health Assessment using machine learning for Prediction), una herramienta de interfaz de usuario que maneja bases de datos de cáncer cervical para detectar patrones y predecir diagnósticos futuros. CHAMP utiliza varios algoritmos de aprendizaje automático para predecir el cáncer cervical con precisión y evaluar procesos para optimizar la predicción. La herramienta, implementada en Python 3.9.0 usando Flask, proporciona una plataforma intuitiva para la detección de patrones. Este enfoque contribuye a la predicción precisa y detección temprana del cáncer cervical mediante el uso de algoritmos de aprendizaje automático y herramientas de validación, promoviendo la toma de decisiones informadas en la atención médica.
Palabras Clave: Cervical cancer, Predictive modeling, Machine learning, XGBoost, Web-based tool, Risk assessment, Early detection
